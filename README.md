# A-B-Testing-Experiment
 Built a complete A/B testing pipeline using Python to evaluate Control vs Treatment performance on user
conversion data.
• Performed hypothesis formulation, data cleaning, and segmentation of experimental groups.
• Computed conversion rates and executed statistical significance tests (Chi-Square and Fisher’s Exact Test).
• Calculated KPI lift, evaluated p-values, and auto-generated experiment recommendations based on statistical
outcomes.
• Demonstrated strong understanding of experimentation frameworks, digital analytics, and data-driven decision
making.
